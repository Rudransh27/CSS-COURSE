Selectors In HTML

Element Selector

The element selector selects HTML elements based on the element name

Id Selector

The id selector uses the id attribute of an HTML element to select a specific element. This id of an element is unique within a page , so the id selector is used to select one unique element!

Class Selector

This class selector selects HTML elements with a specific class attribute. TO select with a specific class, write a period (.) character, followed by class name.
